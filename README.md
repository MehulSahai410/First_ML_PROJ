Student Placement Predictor 

A Machine Learning web application that predicts the likelihood of a student getting placed based on their academic performance (CGPA) and analytical skills (IQ). 

🔴 Live Demo: https://firstmlproj.streamlit.app/

📝 Overview
This project uses a Logistic Regression model trained on a sample student data to classify whether a student has a high or low chance of campus placement. The model is integrated into a clean, interactive web interface.
This project is made by me, a first time machine learning enthusiast to learn more about data preprocessing, data analysis, logistic regression and more
🛠️ Tech Stack
Web Framework:Streamlit
Machine Learning:Scikit-Learn
Data Processing:Pandas, NumPy
Deployment:Streamlit Community Cloud

How to Run Locally

1. Clone the repository:
   git clone https://github.com/MehulSahai410/First_ML_PROJ.git
2. Install the required dependencies:
   pip install -r requirements.txt
3.Run the Streamlit app:
   streamlit run app.py
